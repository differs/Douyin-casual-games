# Douyin Adaptation Points

基于抖音小游戏和 Godot 适配清单，当前客户端后续需要处理的重点：

## 1. 运行时桥接

- 登录
- 请求
- 广告
- 本地存储

建议做法：

- Godot 内维持纯游戏逻辑
- 平台桥接放在独立 JS/GDExtension/原生桥层

## 2. 屏幕与安全区

- 获取设备信息
- 处理胶囊按钮占位
- 处理 DPR / viewport 映射

## 3. 资源与包体

- 控制首包大小
- 大资源走远程或分批加载
- 注意 `.pck` / 导出资源包装方案

## 4. 文件系统

- 包内资源只读
- 存档走 `user://`

## 5. 联调观测

- 屏幕内状态提示
- 关键事件埋点
- 后端日志
